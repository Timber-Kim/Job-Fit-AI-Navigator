# Job-Fit-AI-Navigator

> **어떤 AI를 써야 할지 모르겠다면? 당신의 직무와 상황만 알려주세요.**

## 📖 프로젝트 소개 (About)
**Job-Fit AI Navigator**는 쏟아지는 AI 서비스 홍수 속에서, 사용자의 **직무(Job)**, **상황(Situation)**, **원하는 결과물(Output)**에 딱 맞는 최적의 AI 도구를 추천해 주는 프로젝트입니다. 

단순히 인기 있는 툴을 나열하는 것이 아니라, 실무에서 "당장 보고서를 써야 할 때", "마케팅 이미지를 급하게 만들어야 할 때"와 같이 구체적인 페인 포인트(Pain Point)를 해결하는 것을 목표로 합니다.

## 🚀 주요 기능 (Features)
- **직무별 큐레이션**: 마케터, 개발자, 기획자, 디자이너 등 직군별 최적화 툴 분류
- **상황별 매칭**: 아이디어 도출, 초안 작성, 디자인 생성, 데이터 분석 등 상황에 따른 추천
- **워크플로우 제안**: 단일 툴 추천을 넘어, 여러 AI를 조합해 최고의 효율을 내는 조합법(Recipe) 제공

## 📂 추천 목록 예시 (Preview)

| 상황 (Situation) | 추천 도구 (Tool) | 활용 팁 (Tip) |
| :--- | :--- | :--- |
| **해외 시장 리서치** | **Perplexity** | 출처가 명시된 답변을 제공하므로 팩트 체크에 유리합니다. |
| **사업 제안서 초안** | **Claude 3.5 Sonnet** | 논리적인 글쓰기와 문맥 파악 능력이 뛰어나 초안 작성에 탁월합니다. |
| **발표용 PPT 생성** | **Gamma** | 텍스트 개요만 넣으면 디자인된 슬라이드를 자동 생성합니다. |

*더 많은 데이터는 `data/ai_tools_list.csv`에서 확인하실 수 있습니다.*
